Poetry: Python套件管理器

Poetry 是一個強大的 Python 套件管理工具，它可以幫助你更輕鬆地管理和組織 Python 專案的依賴，並提供一個一致的開發環境，更有效地管理專案，並確保依賴的一致性和可靠性。

      pyproject.toml只會記錄套件本體，剩餘的依賴套件不會被記錄唷！
      等於裡面只會記錄你主動安裝的套件。

    2. poetry.lock 與更新順序

      更新pyproject.toml時，專案中的poetry.lock也會變動，它就像 pip 的requirements.txt，紀錄以安裝的套件與版本。

      poetry add後步驟為：

      1. 更新pyproject.toml。
      2. 依照pyproject.toml的內容，更新poetry.lock。
      3. 依照poetry.lock的內容，更新虛擬環境。

      由此可見，poetry.lock會基於pyproject.toml同步更新。

  2. 更新 poetry.lock

Poetry 常用指令清單

Poetry 的常用指令主要有下面幾個：

  • poetry add (安裝套件)
  • poetry remove (移除套件)
  • poetry export (匯出Poetry)
  • poetry env use (建立虛擬環境指定Python版本)
  • poetry shell (進入Poetry虛擬環境)
  • poetry show (顯示套件)
  • poetry init (初始化)
  • poetry install (安裝)
  • poetry update (更新lock)

在別台主機上重現專案的 Poetry 虛擬環境

要在其他主機上Poetry 虛擬環境其實非常簡單！

  1. 使用poetry shell進入虛擬環境。
  2. 使用poetry install依照poetry.lock記載的套件版本安裝到虛擬環境中。
